# 🕒 Analog Clock

A simple and responsive **Analog Clock** built using **HTML, CSS, and JavaScript**. The clock displays the current system time and updates every second with smooth hand movements.

## 🌐 Live Demo

👉 https://mmuusskkaann.github.io/analog-clock/

## ✨ Features

- 🕒 Real-time analog clock
- ⏱️ Smooth movement of hour, minute, and second hands
- 🎨 Clean and modern user interface
- 📱 Responsive design
- ⚡ Built using pure HTML, CSS, and JavaScript
- 🚫 No external libraries or frameworks

## ⚙️ How It Works

- Retrieves the current system time using JavaScript's `Date` object.
- Updates the clock every second using `setInterval()`.
- Rotates the clock hands using CSS `transform: rotate()`.
- Calculates hand positions based on the current hour, minute, and second.
